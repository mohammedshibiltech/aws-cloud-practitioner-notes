*   **Secure Configurations:** Exploring the transition from relying on local `.env` files to securely pulling environment variables for presentation demos using **AWS Secrets Manager**.

*   **Hosting & Delivery:** Testing deployment pipelines by hosting frontend assets on **Amazon S3**.

## 📂 How to Navigate This Repo

Each module has its own dedicated markdown file in the `/notes` directory containing:
*   **Core Concepts:** Plain-English breakdowns of the services, avoiding marketing jargon.
*   **CLI Commands:** Practical commands for the AWS CLI tested in the terminal.
*   **Architecture & Workflows:** Notes on how different services interact with one another.
*   **Troubleshooting:** Specific configuration roadblocks I hit (like IAM policy errors) and how I resolved them.
